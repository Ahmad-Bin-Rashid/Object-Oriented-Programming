# Object-Oriented Programming (OOP) Coursework

This repository contains all the lab tasks, challenges, and Programming Day (PD) tasks for the Object-Oriented Programming course.

## Repository Structure

The repository is organized into two main categories of folders:

* **Lab `[X]`**: Contains the tasks and challenges completed during the lab sessions. Examples include `Lab 2`, `Lab 3`, `Lab 10`, etc.
* **PD `[X]`**: Contains the Programming Day tasks, assignments, and projects. Examples include `PD 1`, `PD 2`, `PD 12`, etc.

Within each of these folders, you will find independent C# projects (`.sln` and `.csproj` files) for the different tasks (e.g., `task1`, `challenge1`).

## Technologies Used
- C#
- .NET 

## How to Run
1. Navigate to the specific task folder you want to evaluate.
2. Open the `.sln` (Solution) file using Visual Studio.
3. Build and run the project (typically by pressing `F5` or `Ctrl+F5` in Visual Studio).
4. Alternatively, you can navigate into the project directory (where the `.csproj` file is located) and use the .NET CLI: 
   ```bash
   dotnet run
   ```
